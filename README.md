# Agent Systems: Vom Konzept zum eigenen Orchestrator

> Verstehe wie AI Agents funktionieren, wo sie scheitern, und baue dein eigenes System.

## 🎯 Was ist das?

Eine kostenlose, wissenschaftlich fundierte Lernreise zu AI Agent Systems. 15 Module, ~16 Stunden, Deutsch + Englisch. Mit [OpenClaw](https://docs.openclaw.ai) als Open-Source-Referenz-Implementierung.

## 🎓 2 Tracks

- **🟠 Power-User** (~13h): Konfigurieren, Betreiben, Anpassen. Kein Code schreiben.
- **🔴 Developer** (~16h): CLI, Shell-Scripting, eigene Tools und Integrationen.

## 🔬 Wissenschaftliche Basis

18 verifizierte Quellen: ReAct (Yao et al., ICLR 2023), Toolformer (Schick et al., NeurIPS 2023), Generative Agents (Park et al., UIST 2023), MCP Specification (Anthropic, 2024), A2A Protocol (Google, 2025), und mehr.

## 🧵 Roter Faden

Das **Autonomie-Spektrum** — von HITL-intensiv bis vollautonom. Human-in-the-Loop ist kein einzelnes Modul, sondern Design-Prinzip in jedem Modul.
