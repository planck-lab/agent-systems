Wissenschaftliche GrundlageScientific Foundation

🏘️ Park et al. (2023): Generative Agents

25 autonome Agents in einer Sandbox-Welt — jeder mit eigenem Memory-Stream. Drei Kernmechanismen: Aufzeichnen (alles wird gespeichert), Retrieval (relevante Erinnerungen abrufen), Reflection (Erinnerungen zu höheren Einsichten verdichten). Das Ergebnis: Emergentes soziales Verhalten — Agents organisieren selbstständig Partys, verbreiten Gerüchte, verlieben sich. Die drei Komponenten sind direkt auf unsere Systeme übertragbar.25 autonomous agents in a sandbox world — each with their own memory stream. Three core mechanisms: Recording (everything is saved), Retrieval (recall relevant memories), Reflection (distill memories into higher-level insights). The result: emergent social behavior — agents independently organize parties, spread rumors, fall in love. The three components are directly transferable to our systems.

📚 Park et al. (2023): "Generative Agents: Interactive Simulacra of Human Behavior", UIST 2023 Tier A

📚 Packer et al. (2023): MemGPT

Hierarchisches Memory überwindet Kontext-Limits: Wie ein Betriebssystem den Arbeitsspeicher (RAM) und die Festplatte (Disk) verwaltet, unterscheidet MemGPT zwischen Main Context (aktiv, im Kontext-Fenster) und External Storage (archiviert, bei Bedarf abgerufen). Der Agent entscheidet selbst, wann er Informationen ein- und auslagert.Hierarchical memory overcomes context limits: Like an operating system managing RAM and disk, MemGPT distinguishes between main context (active, in context window) and external storage (archived, retrieved on demand). The agent decides on its own when to swap information in and out.

📚 Packer et al. (2023): "MemGPT: Towards LLMs as Operating Systems", arXiv:2310.08560 Tier B
