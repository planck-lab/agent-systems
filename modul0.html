JSON & YAML verstehenUnderstanding JSON & YAML

Agent-Konfigurationen nutzen JSON und YAML. Du musst sie lesen und editieren können:Agent configurations use JSON and YAML. You need to read and edit them:

JSON 
{
  "name": "Planck",
  "model": "sonnet",
  "tools": ["read", "write"]
}
YAML 
name: Planck
model: sonnet
tools:
  - read
  - write

3
Node.js & npm installierenInstall Node.js & npm

OpenClaw läuft auf Node.js. Installiere die aktuelle LTS-Version:OpenClaw runs on Node.js. Install the current LTS version:

# Mac (mit Homebrew)
brew install node

# Oder via nvm (empfohlen)
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
nvm install --lts

# Prüfen
node --version    # z.B. v22.x.x
npm --version     # z.B. 10.x.x
✅ node --version zeigt eine Version ≥18shows a version ≥18

npm i -g @anthropic/openclaw

# Setup starten
openclaw onboard

Der Onboarding-Wizard fragt dich nach:The onboarding wizard will ask for:

  1. API-Key (Anthropic oder OpenAI)API Key (Anthropic or OpenAI)
  2. Bevorzugtes Modell (Haiku für günstig, Sonnet für besser)Preferred model (Haiku for cheap, Sonnet for better)
  3. Channel-Auswahl (Telegram empfohlen)Channel selection (Telegram recommended)

5
Telegram-Bot erstellenCreate Telegram Bot
  1. Öffne Telegram und suche @BotFatherOpen Telegram and search for @BotFather
  2. Sende /newbot und folge den AnweisungenSend /newbot and follow the instructions
  3. Kopiere den Bot-TokenCopy the bot token
  4. Füge den Token in OpenClaw ein: openclaw config set telegram.token YOUR_TOKENAdd the token to OpenClaw: openclaw config set telegram.token YOUR_TOKEN
  5. Starte den Gateway: openclaw gateway startStart the gateway: openclaw gateway start
  6. Schreibe deinem Bot „Hallo" auf TelegramSend your bot "Hello" on Telegram
✅ Dein Agent antwortet auf „Hallo" in TelegramYour agent responds to "Hello" in Telegram

🔧 Troubleshooting🔧 Troubleshooting

Problem: „command not found: openclaw"Problem: "command not found: openclaw"

→ Terminal neu starten oder export PATH="$PATH:$(npm root -g)/.bin" ausführen→ Restart terminal or run export PATH="$PATH:$(npm root -g)/.bin"

Problem: Node.js Version zu altProblem: Node.js version too old

→ nvm install --lts && nvm use --lts→ nvm install --lts && nvm use --lts

Problem: API-Key wird nicht akzeptiertProblem: API key not accepted

→ Prüfe auf Tippfehler. Key muss mit „sk-ant-" (Anthropic) oder „sk-" (OpenAI) beginnen.→ Check for typos. Key must start with "sk-ant-" (Anthropic) or "sk-" (OpenAI).

Problem: Telegram-Bot antwortet nichtProblem: Telegram bot doesn't respond

→ openclaw gateway status prüfen. Läuft der Gateway? Token korrekt?→ Check openclaw gateway status. Is the gateway running? Token correct?

Problem: „Permission denied" bei npm install -gProblem: "Permission denied" on npm install -g

→ Nutze nvm statt system-npm, oder sudo npm i -g @anthropic/openclaw→ Use nvm instead of system npm, or sudo npm i -g @anthropic/openclaw
