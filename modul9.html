Multi-Agent: Kern-Pattern 4/4 (Ng, 2024)Multi-Agent: Core Pattern 4/4 (Ng, 2024)

Andrew Ng definiert Multi-Agent als viertes und komplexestes der agentic Design Patterns. Die Idee: Spezialisierte Agents für Teilaufgaben — wie ein Team, in dem jeder seine Stärke einbringt.Andrew Ng defines multi-agent as the fourth and most complex agentic design pattern. The idea: Specialized agents for sub-tasks — like a team where everyone contributes their strength.

Wu et al. (2023, AutoGen) formalisieren dies als ConversableAgent-Pattern: Agents konversieren miteinander, lösen Teilaufgaben und geben Ergebnisse weiter. Hong et al. (2024, MetaGPT) gehen einen Schritt weiter mit rollen-basierter Koordination — Agents übernehmen spezifische Rollen (Researcher, Coder, Reviewer) in einem strukturierten Workflow.Wu et al. (2023, AutoGen) formalize this as the ConversableAgent pattern: agents converse with each other, solve sub-tasks, and pass on results. Hong et al. (2024, MetaGPT) go a step further with role-based coordination — agents take specific roles (Researcher, Coder, Reviewer) in a structured workflow.

📚 Ng (2024): "Agentic Design Patterns" Tier C · Wu et al. (2023): "AutoGen", arXiv:2308.08155 Tier B · Hong et al. (2024): "MetaGPT", ICLR 2024 Tier A

A2A — Agent-to-Agent ProtocolA2A — Agent-to-Agent Protocol

A2A ist für Agents, was HTTP für Webserver ist. Ein Industriestandard (Google, 2025) für die Kommunikation zwischen Agents — auch über Framework-Grenzen hinweg.A2A is for agents what HTTP is for web servers. An industry standard (Google, 2025) for communication between agents — even across framework boundaries.

Vier Grundmuster, wie Agents zusammenarbeiten:Four basic patterns for how agents collaborate:

➡️
Sequential
Agent A → Agent B → Agent C. Jeder baut auf dem Vorgänger auf.Agent A → Agent B → Agent C. Each builds on the predecessor.
Research → Zusammenfassung → FormatierungResearch → Summary → Formatting
⚡
Concurrent
Agent A ∥ Agent B ∥ Agent C → Merge. Parallele Bearbeitung, danach Synthese.Agent A ∥ Agent B ∥ Agent C → Merge. Parallel processing, then synthesis.
3 Quellen parallel recherchieren → zusammenführenResearch 3 sources in parallel → merge
🤝
Handoff
Agent A delegiert Teilaufgabe an Agent B mit vollem Kontext.Agent A delegates sub-task to Agent B with full context.

Orchestrator-Worker
